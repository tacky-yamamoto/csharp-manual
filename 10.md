# マウス入力

## 座標の取得方法
```cs
    // 2次元クラスPoint型の変数mpを宣言
　　Point mp = MousePosition

    // cposに、マウスのフォーム座標を取り出す。
    //// MousePositionにマウス座標のスクリーン左上からのX、Yが入っている。
    //// PointToClient(mp)を使うと、スクリーン座標を、フォーム座標に変換できる。
    mp = PointToClient(mp);

    // ラベルに座標を表示
    //// 変換したフォーム座標は、mp.X、mp.Yで取り出せる。
    label2.text = ""+mp.X+","+mp.Y;
```
